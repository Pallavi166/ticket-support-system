# ticket-support-system
.Net Ticket Support System Assignment


## Overview
This project is a Ticket Support System built using ASP.NET Core Web API and Windows Forms.  
It allows users to create and manage support tickets, assign them to users, update ticket status, and add comments.

## Technologies Used
- C#
- ASP.NET Core Web API
- Windows Forms (.NET)
- MySQL
- Entity Framework Core
- Swagger

## Features
- User Login
- Create Ticket
- View Ticket List
- Assign Ticket to User
- Update Ticket Status
- Add Comments to Ticket
- View Ticket Details
- Ticket Status History

## Project Structure
TicketSupport.API – Backend API built with ASP.NET Core  
TicketSupport.Desktop – Windows Forms desktop application

## How to Run the Project

### Run the API
1. Open TicketSupport.API in Visual Studio
2. Run the project
3. Swagger will open to test APIs

### Run the Desktop Application
1. Open TicketSupport.Desktop
2. Run the Windows Forms application
3. Login and manage tickets

## Database
MySQL database is used to store:
- Users
- Tickets
- Ticket Comments
- Ticket Status History
